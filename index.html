¿Qué son las inecuaciones?

Las inecuaciones son expresiones matemáticas que establecen una relación de desigualdad entre dos cantidades. A diferencia de las ecuaciones, las inecuaciones comparan dos expresiones usando los signos <, >, ≤ o ≥ para determinar si una cantidad es mayor, menor o igual que otra.

Clases / Tipos

Inecuaciones lineales

Son aquellas en las que la variable tiene exponente 1. Ejemplo: 2x + 3 < 7.

Inecuaciones cuadráticas

Contienen la variable elevada al cuadrado. Ejemplo: x² - 4x + 3 > 0.

Inecuaciones racionales

Son aquellas que involucran fracciones con variables en el denominador.

Ejemplos

Ejemplo 1: Resolver 3x - 5 < 10

Solución: 3x < 15 → x < 5

Ejemplo 2: Resolver 2x + 1 ≥ 7

Solución: 2x ≥ 6 → x ≥ 3

Problemas para Resolver

Problema 1: Resolver la inecuación: 5x - 2 ≥ 8

5x - 2 ≥ 8 → 5x ≥ 10 → x ≥ 2

Problema 2: Resolver: 4 - x < 1

4 - x < 1 → -x < -3 → x > 3

Problema 3: Resolver: x² - 3x + 2 > 0

Factorizando: (x - 1)(x - 2) > 0. Solución: x < 1 o x > 2.
